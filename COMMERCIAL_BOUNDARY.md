# Commercial boundary

This repository is an early-access offer and validation shelf.

Allowed here:

- Public product previews.
- Generic purchase-interest issues.
- Generic scope feedback without private data.

Not allowed here:

- Secrets or credentials.
- Private logs.
- Customer data.
- Proprietary source code.
- IP addresses or infrastructure topology.
- Production exports.
- Emergency support requests.
- Legal/compliance/security certification requests.

Full paid ZIP kits are not stored in this public repository. Delivery/payment will be handled separately by the owner once a payment path is available.
